# 打鍵トレーナーの仕様

## 打鍵トレーナーで出題される単語

### 高速タイピング

[打鍵トレーナー（高速タイピング）](http://typing.tsurizamurai.com/daken_010/type.cgi)で出題される文章は以下の18個。

1. 生活していく中でインターネットはなくてはならない存在になりました
2. 彼なら完走する可能性はあるがこの調子なら時間がかかりそうだ
3. 正当な手段を選択するには資料の詳細を参考にして相談しよう
4. 彼は商業都市について調査要請を依頼していたようだ
5. その少量の装置を調整したければ課長に相談してください
6. 校長先生と教頭先生は今年に定年退職するそうだ
7. 高速タイピングはサクサクと楽勝すぎて何度も挑戦する
8. 小学生は行政法を解読したことにワクワクした
9. 大学までの通学時間が異様に長いのは気のせいだろうか
10. 長文で父さんとの勝負に勝利するため早朝から挑戦する
11. 消防士になるためには相当な苦労が必要とされています
12. 大切な話を伝えるためにはお互い相手の目を見て話をしよう
13. 何気なく人間は暮らしているが環境は毎日汚染されている
14. 面接対策には前もってやる擬似面接を怠らないようにしましょう
15. 視力が低下してきた感じがしたためコンタクトを使用することにしました
16. あの映画を鑑賞したいなら監督からの承諾を得た方がいいよ
17. 町長さんが市長に相談したらそれに市長は応じてくれなかった
18. 将来の健康のためにジョギングをすることはすばらしいと感じています
